## Artifacts Produced

### app/views/engagements.js (new file, 410 lines)

**Exports:**
- `export function render(activeTab, param)` — returns container div with h2 "New Engagement", subtitle "Create a new project or service engagement.", and the complete Create Engagement form card. Same fields as proposals.js: engagement type radio buttons, client picker, name, location, budget, contract cost, personnel picker, Create Engagement button.
- `export async function init(activeTab, param)` — registers 6 window functions, initializes pill container, starts clients + users onSnapshot listeners.
- `export async function destroy()` — unsubscribes both listeners, deletes all 6 window functions, resets all module state.

**Imports:**
- Firebase: `db, collection, onSnapshot, query, where`
- Utils: `showLoading, showToast, syncPersonnelToAssignments, syncServicePersonnelToAssignments, escapeHTML`
- Engagement: `createEngagement` from `../engagement-create.js`
- Pill wrappers from `./proposals.js`: `renderProposalPillsFor`, `proposalFilterPersonnelDropdownFor`, `proposalSelectPersonnelIn`, `proposalRemovePersonnelFrom`, `proposalShowPersonnelDropdownFor`

**Window closures in init() (adapter pattern):**
```javascript
window.proposalSelectPersonnel = (userId, userName) => {
    selectedPersonnel = proposalSelectPersonnelIn(
        userId, userName, selectedPersonnel,
        () => renderProposalPillsFor(selectedPersonnel)
    );
};
window.proposalRemovePersonnel = (userId) => {
    selectedPersonnel = proposalRemovePersonnelFrom(
        userId, selectedPersonnel,
        () => renderProposalPillsFor(selectedPersonnel)
    );
};
window.proposalShowPersonnelDropdown = () =>
    proposalShowPersonnelDropdownFor(selectedPersonnel, usersData);
window.proposalFilterPersonnelDropdown = (text) =>
    proposalFilterPersonnelDropdownFor(text, selectedPersonnel, usersData);
```
