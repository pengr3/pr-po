# Phase 97.2 Plan 01: Iteration Rail UX Overhaul — Filter, Timestamps, Delete Summary

**One-liner:** Auto-snapshot suppression from the visible rail with relative timestamps and per-card delete buttons backed by an inline confirm modal.

## What Was Built

**Task 1 — JS overhaul (`app/views/project-plan.js`):**
- Added `_formatIterTimestamp(savedAt)` helper immediately before `renderIterRail()`. Returns `'Just now'` for null/non-Timestamp, `'< 1h ago'` for diff < 3600000ms, `'Xh ago'` for diff < 86400000ms, `toLocaleDateString()` for older.
- `renderIterRail()`: computes `const visible = _iterations.filter(i => !i.auto)` and drives badge count, empty-state guard, and render loop from `visible` instead of `_iterations`.
- Dot numbers now use `visible.length - idx` for clean sequential numbering of only non-auto items.
- Removed `iter.auto ? 'auto' : ''` class on `.iter-tl-dot` and `${iter.auto ? ' <span class="iter-auto-tag">auto</span>' : ''}` chip from `.iter-tl-name`.
- Added `<button class="iter-tl-delete-btn" onclick="window.deleteIteration('${iter.id}')" title="Delete iteration">&#xD7;</button>` inside `.iter-tl-actions`.
- Added `async function deleteIteration(iterationId)`: finds iter in `_iterations`, builds an inline confirm modal (same `document.createElement` pattern as `openIterConfirm()`), uses Cancel/Delete buttons with addEventListener (not onclick attributes — avoids inline handler complexity for async flow). On confirm: calls `deleteDoc(doc(db, 'project_iterations', iterationId))`, then `loadIterations()` + `renderIterRail()`. Wrapped in try/catch with `showToast('Failed to delete iteration.', 'error')`.
- `window.deleteIteration = deleteIteration` added to init() Phase 97 block.
- `delete window.deleteIteration` added to destroy() Phase 97 cleanup block.

**Task 2 — CSS cleanup (`styles/views.css`):**
- Deleted `.iter-tl-dot.auto` block (5 lines — dead after auto filter).
- Deleted `.iter-auto-tag` block (7 lines — dead after chip removal).
- Added `.iter-tl-delete-btn` rule: `padding:3px 6px; font-size:11px; font-weight:700; border:none; border-radius:4px; background:#fee2e2; color:#ef4444; cursor:pointer; margin-left:auto`.
- Added `.iter-tl-delete-btn:hover` rule: `background:#ef4444; color:#fff`.
- Added `align-items:center` to `.iter-tl-actions` so the delete button aligns vertically with the other buttons.
