## Changes Made

### Task 1 — app/views/project-plan.js

Added `const today = new Date().toISOString().slice(0, 10);` once inside `renderGantt()` before `const frappeTasks = [];` so it is available to `appendNode()` via closure.

In `appendNode()`, appended an `else` branch after `else if (isParent)`:

```javascript
else {
    // Phase 86.11 — leaf task bar coloring by status
    const status = computeStatus(t, today);
    const statusClassMap = { overdue: 'bar-status-overdue', complete: 'bar-status-complete', 'not-started': 'bar-status-not-started' };
    customClass = statusClassMap[status] || '';
}
```

Milestone and parent branches are untouched. `computeStatus()` from Plan 01 reused with no duplication.
