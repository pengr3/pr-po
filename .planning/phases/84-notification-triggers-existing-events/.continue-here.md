## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| batch.set+merge for nested fields | `batch.set(ref, {'a.b.c': v}, {merge:true})` writes a literal top-level key named "a.b.c" — the nested structure is untouched (silent failure). | advisory | Always use `batch.update(ref, {'a.b.c': v})` for partial nested writes. Fix already committed in 4481569. |
