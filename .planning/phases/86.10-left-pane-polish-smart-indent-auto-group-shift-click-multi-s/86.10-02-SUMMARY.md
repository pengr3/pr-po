# Phase 86.10-02: Shift+click multi-select + group drag Summary

**Shift+click selects a contiguous row range highlighted blue; dragging any selected row moves the entire group as a block**

## Performance

- **Duration:** ~8 min
- **Tasks:** 2 (+ human-verify checkpoint pending)
- **Files modified:** 2

## Accomplishments
- `_selectedRowIds` Set and `_lastClickedRowId` declared at module scope
- `selectRow()` clears multi-select on plain click; `selectRow(null)` preserves anchor
- `clearSelection()` clears both new vars and removes `tg-multi-selected` from DOM
- `_gridRowClickHandler` shift+click path: computes flattenTreeDepthFirst range, applies `tg-multi-selected` CSS to each row
- `handleGroupDrop()` reorders selected block relative to drop target via `commitRowOrderReorder`
- `_gridDropHandler` routes to `handleGroupDrop` when `_selectedRowIds.size > 1` and dragged row is in selection
- `destroy()` clears `_selectedRowIds` and `_lastClickedRowId`
- `.tg-row.tg-multi-selected` CSS rule added to views.css (blue tint matching tg-selected)
