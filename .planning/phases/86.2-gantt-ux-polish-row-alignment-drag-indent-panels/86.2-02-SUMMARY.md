# Phase 86.2 Plan 02: JS Behavior Defect Fixes Summary

**One-liner:** Five-defect JS fix pass — drag isolation to .tg-rn handle, Enter key commit for Duration/Resources, smart indent inheritance for new trailing rows, and Gantt popup suppression.

## What Was Built

Four targeted edits + two verify-only outcomes to `app/views/project-plan.js`:

1. **DEFECT-3 (Drag isolation):** `_gridDragStartHandler` now uses `e.target.closest('.tg-rn')` as the sole gate. Drag only begins when the user clicks the row-number column. The old INPUT-tag guard is removed — it was insufficient and caused drag-cursor flicker on regular input clicks.

2. **DEFECT-10 (Enter key commits):** New `_gridKeydownHandler` wired in `bindGridEvents` via event delegation. On `Enter`, if the focused input is a `tg-input` on `col=duration` or `col=resources` (not `__new__`), calls `input.blur()` which triggers the existing `handleGridCellBlur` path. Declared at module scope, cleaned in `destroy()`.

3. **DEFECT-6 (Smart indent inheritance):** `handleNewRowCommit` now computes `inheritedParentId` from `sortedForInherit[last].parent_task_id` before building `docData`. New trailing-row tasks inherit the indent level of the task immediately above them instead of always being placed at root level.

4. **DEFECT-4 (Gantt popup suppression):** `popup: false` added to the Gantt constructor options object. Suppresses Frappe Gantt's default click-triggered bar popup.

5. **DEFECT-2 (Back button — verify only):** `href="#/projects"` confirmed present on `.plan-back-link` at render() line 47. Correct — no code change needed.

6. **DEFECT-5 (Monthly header — verify only):** Frappe Gantt v1.2.2 Month mode renders full month names (e.g. "January") in the lower header and year in the upper header. Already matches MS Project Month view. No code change needed.
