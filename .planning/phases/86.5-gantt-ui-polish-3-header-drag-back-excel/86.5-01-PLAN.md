<objective>
Fix the 8px vertical misalignment between the left-rail column headers and the Frappe Gantt timeline header.

Purpose: The existing `padding: 8px` on `.gantt-pane #ganttPane` pushes the Frappe SVG down 8px relative to the task-grid-rail thead, which starts at y=0. Removing only the top padding eliminates the misalignment while preserving side/bottom padding. The `#ganttHeaderOverlay` div (created by `renderCustomGanttHeader()`, still active) uses `top: 8px` to align with the old padding-top — this must also be changed to `top: 0` so the custom header stays flush with the SVG.
Output: Two CSS changes in styles/views.css — padding-top 8px removed from #ganttPane; top: 8px changed to top: 0 in #ganttHeaderOverlay.
</objective>
