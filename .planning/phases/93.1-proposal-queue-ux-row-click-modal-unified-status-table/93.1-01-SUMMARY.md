# Phase 93.1 Plan 01: Queue Row-Click Handler Summary

Row-click affordance added to the Proposal Approval Queue table in `_renderHomeApprovalQueueHtml`: clicking any queue row (except the action buttons) now opens the proposal detail modal via `window.openProposalModal`.

## What Was Built

Modified `_renderHomeApprovalQueueHtml` in `app/views/home.js`:

1. Each `<tr>` now has `onclick="window.openProposalModal && window.openProposalModal('${escapeHTML(p.id)}')"` using the established `&&` safety guard.
2. Each `<tr>` has `style="cursor:pointer;"` plus `onmouseenter`/`onmouseleave` toggling `style.background` to `#f8fafc` on hover.
3. Both Approve and Reject button `onclick` handlers now begin with `event.stopPropagation();` so they fire independently without bubbling to the row-click handler.

No other functions in `home.js` were touched.
