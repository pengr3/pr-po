| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-97-04-02 | Tampering (XSS) | openIterConfirm() modal innerHTML | mitigate | escapeHTML(iter.label) before inserting label into modal HTML |
| T-97-04-03 | Tampering (XSS) | showUndoToast() innerHTML | mitigate | escapeHTML(msg) applied to message passed to toast |
