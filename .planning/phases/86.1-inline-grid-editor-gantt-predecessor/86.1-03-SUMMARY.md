# Phase 86.1 Plan 03: Resource Names Assignee Picker Summary

**One-liner:** Wired the Resource Names cell click stub from Plan 01 to a fully-functional inline pill picker — position:fixed popup anchored below the cell, toggleable personnel pills, each toggle an atomic Firestore write, outside-click dismissal.

## What Was Built

### Task 1: Add openAssigneePicker function (commit 6df73dd)

Lines added to `app/views/project-plan.js` (+67 lines net):

**`openAssigneePicker(taskId, anchorEl)`** (after computeDepth helper, before Gantt section):
- Removes any existing `#taskAssigneePicker` first (idempotent — prevents DOM accumulation, T-86.1.3-06)
- Guard: `if (!taskId || taskId === '__new__') return;` — belt-and-suspenders against empty trailing row
- Looks up task in `tasks[]`; returns if not found
- Calls `normalizePersonnel(currentProject)` → `{ names, userIds }`
- `currentAssignees = Array.isArray(t.assignees) ? [...t.assignees] : []` — defensive copy
- Creates `<div id="taskAssigneePicker">` at `position:fixed` anchored to `anchorEl.getBoundingClientRect().left/bottom+4`
- Renders `<span class="personnel-pill${isOn ? ' selected' : ''}" data-uid="...">` for each UID; `escapeHTML` on both uid and name (T-86.1.3-01)
- Empty personnel → `<span style="color:#94a3b8;font-size:13px;">No personnel on this project.</span>`
- Wraps all pills in `<div class="personnel-pills-container">`
- Each pill's click handler: toggles uid in `currentAssignees`, updates `.selected` class, awaits `updateDoc(doc(db,'project_tasks',taskId), { assignees: [...currentAssignees], updated_at: serverTimestamp() })`
- Outside-click: `setTimeout(10ms)` → `document.addEventListener('click', handler)` that removes picker if `!picker.contains(e.target) && e.target !== anchorEl`

**`destroy()` cleanup** (added alongside `taskGridContextMenu?.remove()`):
- `document.getElementById('taskAssigneePicker')?.remove();`

**`.personnel-pill.selected` CSS** added to `styles/components.css`:
```css
.personnel-pill.selected {
    background: var(--primary, #1a73e8);
    color: #ffffff;
}
```
