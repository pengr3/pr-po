## Decisions Made

- **Sentinel-attribute de-dup deviation from PATTERNS.md**: PATTERNS.md does not show de-dup logic. Added `barWrapper.dataset.linkBound = '1'` and `svg.dataset.linkBound = '1'` sentinels because `gantt.refresh(tasks)` keeps the same SVG DOM elements across snapshot re-fires (Frappe only rebuilds on `new Gantt()` calls). Without the sentinel, each snapshot would bind a new set of listeners to the same elements, multiplying handler invocations exponentially. Frappe's full rebuild creates a fresh SVG root with no `dataset.linkBound`, so the binding fires cleanly on rebuild too. T-86.1.4-08 mitigation fully satisfied.

- **Call order preserved**: `applyFsViolationStyles()` → `renderTodayLine()` → `initGanttDragLink()`. After a successful drag-link write, the snapshot fires `renderGantt()` which re-runs `applyFsViolationStyles()` — if the new dep violates FS (target starts before source ends), `checkAndToastFsViolations()` fires its warning toast. Order is correct.

- **detectDependencyCycle reuse matches Plan 01 Predecessors-column pattern**: Both paths call `detectDependencyCycle(newDeps, toTaskId)` before committing. The cycle detector in Phase 86 operates on the full `tasks[]` array with a DFS pass — same behavior for drag-to-link as for Predecessors cell blur.

### Deviations from PATTERNS.md

**1. [Improvement - De-dup guard] Sentinel-attribute de-dup not in PATTERNS.md**
- **What:** PATTERNS.md SVG overlay pattern excerpt (lines 729-839) does not include `data-linkBound` sentinel checks
- **Change:** Added `if (barWrapper.dataset.linkBound === '1') return; barWrapper.dataset.linkBound = '1';` per-bar-wrapper and `if (svg.dataset.linkBound !== '1') { svg.dataset.linkBound = '1'; ...}` per SVG root
- **Why:** Without de-dup, `initGanttDragLink()` called on every snapshot re-fire would accumulate multiple `mouseenter/mouseleave/mousedown` listeners on the same `.bar-wrapper` elements that `gantt.refresh()` keeps in place. The SVG-level `mousemove` and `mouseup` would also multiply. This was specified in the plan's Task 1 action item #4 as a required deviation.
- **Impact:** No behavior change for the first bind. Subsequent binds (snapshot re-fires with `gantt.refresh()`) skip re-binding silently. Frappe full rebuild creates fresh DOM → fresh sentinel → fresh bind. All paths bounded per T-86.1.4-08.
