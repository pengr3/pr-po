# Phase 65.9 Plan 01: Integrate Delivery Fees RFP Support Summary

Delivery fee RFP flow integrated into PO Tracking: right-click context menu extended with conditional "Request Delivery Fee Payment" option, simplified delivery fee modal created, and 8px colored dot indicator added to PO ID cells showing delivery fee payment status.

## What Was Built

### Task 1: Context Menu + Modal + Submit Function

**showRFPContextMenu() extended:** Now finds the PO from `poData`, checks `po.delivery_fee > 0`, and conditionally renders a second menu item "Request Delivery Fee Payment". The item is hidden when no delivery fee exists, grayed out with "(RFP exists)" suffix when a Delivery Fee RFP already exists for that PO (checked via `rfpsByPO[po.po_id]?.some(r => r.tranche_label === 'Delivery Fee')`).

**openDeliveryFeeRFPModal(poDocId):** Simplified RFP creation modal matching the visual design of `openRFPModal` but with no tranche dropdown. Amount field displays `formatCurrency(po.delivery_fee)` as readonly. Includes full bank transfer fields with alt bank section. Double-checks dedup before opening. Submit button calls `window.submitDeliveryFeeRFP`.

**submitDeliveryFeeRFP(poDocId):** Same validation flow as `submitRFP` (invoice number, due date, payment mode required; bank fields required for Bank Transfer; Other mode requires specification). Writes rfps document with `tranche_label: 'Delivery Fee'`, `tranche_percentage: 0`, `amount_requested: po.delivery_fee`. Uses existing `generateRFPId(po.po_id)` for ID generation.

Window functions registered in `attachWindowFunctions()` and cleaned up in `destroy()`.

### Task 2: Delivery Fee Status Dot

Before each PO row template, computes `deliveryFeeDotHtml`:
- If `po.delivery_fee <= 0`: empty string (no dot)
- If `po.delivery_fee > 0`: filters `rfpsByPO[po.po_id]` for Delivery Fee RFPs, checks if any have `payment_records` covering `amount_requested`
- Dot color: `#059669` (green) when paid, `#ef4444` (red) when not yet paid or no RFP submitted
- Positioned `absolute; top:4px; right:4px; width:8px; height:8px; border-radius:50%; z-index:2`
- Title tooltip: "Delivery Fee: PHP {amount} — Paid" or "Delivery Fee: PHP {amount} — Not yet paid"
- Injected into `po-id-cell` td between the payment fill div and the PO ID span
