<!-- Batched write pattern from wipe.js (adapted for set instead of delete): -->

```javascript
const BATCH_SIZE = 500;

async function batchWrite(db, collectionName, records) {
    let written = 0;
    for (let i = 0; i < records.length; i += BATCH_SIZE) {
        const batch = db.batch();
        records.slice(i, i + BATCH_SIZE).forEach(record => {
            const ref = db.collection(collectionName).doc();
            batch.set(ref, record);
        });
        await batch.commit();
        written += Math.min(BATCH_SIZE, records.length - i);
    }
    return written;
}
```

**CSV Parsing (no external library — built-in Node.js only):**
Implement a `parseCSV(filePath)` function:
1. Read file with `fs.readFileSync(filePath, 'utf8')`
2. Handle BOM (byte order mark): strip leading `\uFEFF` if present (common in Excel CSV exports)
3. Split by newlines (`\r\n` or `\n`)
4. First line = headers. Trim each header. Normalize to lowercase with underscores (e.g., "Client Code" → "client_code", "Company Name" → "company_name"). This makes the script flexible to header naming as specified in CONTEXT.md.
5. For each subsequent non-empty line, split by comma. Handle basic quoted fields: if a field starts with `"`, find the closing `"` and treat everything between as one field (commas inside quotes are not delimiters). Trim whitespace from each value.
6. Return `{ headers: string[], rows: object[] }` where each row is `{ [header]: value }`.
7. Each row object also gets a `_rowNum` property (1-based, counting from the first data row after headers) for error reporting.

**Header normalization function `normalizeHeader(header)`:**

function normalizeHeader(header) {
