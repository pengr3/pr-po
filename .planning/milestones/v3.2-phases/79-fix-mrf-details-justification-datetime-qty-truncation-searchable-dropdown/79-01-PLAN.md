
Insert the following two cells immediately before the closing `</div>` of the grid (line 3066), wrapped in a conditional so they only appear when `!isNew`:

```javascript
${!isNew ? `
            <div>
                <div style="font-size: 0.75rem; color: #5f6368;">Date Submitted</div>

            </div>

```
