<objective>
Mobile-optimize the Financial Breakdown Modal (#expenseBreakdownModal in app/expense-modal.js).

Purpose: The modal contains three rows of inline-styled grid scorecards (2/3/2 columns), a tab bar with 3 inline-flex buttons, and tables inside collapsible category cards. On 375px viewports the 3-column scorecard row squeezes each card to ~90px effective width — amounts wrap ugly, labels clip. Tab bar may clip. Category item tables overflow. Fix is to replace inline grid styles with CSS classes that have mobile overrides, add a scroll rule for category items (with `min-width: 400px` on the inner table per peer review MEDIUM #1 so the scroll actually triggers), and allow the tab bar to scroll horizontally when needed.

Output:
- styles/views.css — new Phase 73.2 block appended after the Phase 73.1 Finance block (closes at line 1812)
- app/expense-modal.js — replace inline `style="display: grid; grid-template-columns: ..."` on 3 scorecard rows and 1 tab nav div with CSS classes

No data logic changes. No new files. Pure presentation.
</objective>

    ```css
    /* ========================================
       FINANCE MODALS — MOBILE RESPONSIVE (Phase 73.2)

        gap: 1rem;
        margin-bottom: 1.5rem;

    .em-scorecard-2col.em-row3 {
        margin-bottom: 2rem;
    }

    /* Tab bar container: horizontal scroll when tabs would clip */
    .em-tab-bar {
        overflow-x: auto;
        -webkit-overflow-scrolling: touch;
        border-bottom: 2px solid #e5e7eb;
        white-space: nowrap;
    }

    .em-tab-bar .expense-tab {
        white-space: nowrap;
    }

    /* Category card item tables: horizontal scroll when content is wider than card.
       Per peer review (Gemini MEDIUM #1): inner table needs min-width so that
       overflow-x actually triggers scroll instead of letting cells wrap vertically. */
    #expenseBreakdownModal .category-items {
        overflow-x: auto;
        -webkit-overflow-scrolling: touch;
    }

    #expenseBreakdownModal .category-items table {
        min-width: 400px;
    }

    @media (max-width: 768px) {
        /* Collapse scorecard grids to a single column on mobile */
        .em-scorecard-2col,
        .em-scorecard-3col {
            grid-template-columns: 1fr;
        }
    }
    ```
