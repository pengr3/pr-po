## Setup

1. Open `http://localhost:8000/` in a browser logged in as super_admin or operations_admin (Firestore rules require it for `create` and `delete` on baselines — only `create` is exercised here).
2. Navigate to **Projects** → pick a project that has **at least 3 tasks with start + end dates**. If none exists, create one with 3 dummy tasks dated this week.
3. Click into the project → **Plan** route (`#/projects/{code}/plan`).
4. Hard-reload (Ctrl+Shift+R) to bust any cached project-plan.js from before the fix.
