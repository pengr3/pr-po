## Requirements
- Must work without a build step (no webpack/vite â€” pure static SPA)
- Detection must be passive (polling, no user action needed to trigger check)
- Poll interval: **30 minutes** (deployed ~twice/month; HEAD request is headers-only, zero Firebase usage)
- Notification must be dismissible without forcing a refresh
- Must not interfere with active workflows (e.g. filling in a form)
- UX: **Variant A â€” fixed top strip** (full-width, slides in above nav, stacks vertically on mobile)
- Notification rows: **3-line anatomy** â€” event title + optional "â—Ź Action needed" chip + relative time / objectId Â· objectName / actor name (omit if System)
- Relationship badge: **not used** â€” noise without proportional value
- FYI chip: **not used** â€” silence signals informational
- Schema delta: only **2 new fields** needed â€” `object_name` (embedded from source doc) and `actor_name` (embedded from auth.currentUser or "System")
